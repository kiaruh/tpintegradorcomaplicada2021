1) Con Amazon Web Services. Servidor web estándar en AWS

Crear un servidor web

• Actualizar el servidor.

• Instalar Apache (nombre del paquete: apache2).

• Crear, en la carpeta /var/www/html, un archivo llamado trabajo.html y que contenga, como contenido, la opinión personal sobre la materia.

• Instalar una base de datos postgresql (paquete postgresql).

• Desplegar una página web de ejemplo (puede decir lo que se prefiera).

• Configurar ssh para que permita entrar por usuario y contraseña:

o El usuario es profesor

o Contraseña (número de legajo)

• Dentro de la carpeta raíz del usuario profesor, crear:

o Un link a la carpeta /var/www/html

o Un hard-link al scripting-tp-integrador.sh

Scripting

Se debe generar un script llamado: scripting-tp-integrador.sh, el cual tiene que mostrar un menú con las siguientes funcionalidades:

1) Pedir un número entero y mostrar esa cantidad de elementos de la sucesión de Fibonacci.

2) Pedir una cadena de caracteres y evaluar si es palíndromo o no.

3) Pedir el path a un archivo de texto y mostrar, por pantalla, la cantidad de líneas que tiene el archivo (en caso de no estar, que muestre el mensaje “no existe”).

4) Salir (saludando al usuario que ejecuta el programa).
